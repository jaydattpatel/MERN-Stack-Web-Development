1. Install The gulp command line interface gulp-cli globally :
	npm install -g gulp-cli

2. Install The JavaScript Task Runner gulp as dev dependency:
	npm install gulp --save-dev

3. Install required the gulp plugin as dev dependency:
 	npm i [plugin_Name] --save-dev
