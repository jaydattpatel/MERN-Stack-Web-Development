* content-box: This property is by default and not allow to include padding, border and margin in height and width of content. So, :
[Total Box height = height + top padding + bottom padding + top border + bottom border]
[Total Box width = width + left padding + right padding + left border + right border]
[content height = height]
[content width = width]

Div-1

* border-box: This property is allow to include padding, border and margin in height and width of content. So, :
[Total Box height = height]
[Total Box width = width]
[content height = Total Box height - top padding - bottom padding - top border - bottom border]
[content width = Total Box width - left padding - right padding - left border - right border]
